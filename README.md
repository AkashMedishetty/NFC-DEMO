# NFC Conference Demo Page

Premium mobile-first landing page for NFC marketing demonstration.

## Features

- Smooth expanding circle loader animation
- Center-stage welcome message with float-to-position effect
- Responsive design optimized for mobile devices
- Glass-morphism UI elements
- Professional typography and spacing
- Touch-optimized interactions

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- Framer Motion (animations)
- TailwindCSS (styling)
- Mobile-first responsive design
